Инструкция по запуску приложения Test Shop:

1. Открываем Python IDE (PyCharm)
2. Открываем файл init_sql.py
3. Кликаем ПКМ на любом месте в поле кода. В выпадающем меню кликаем "Run init_sql.py"
4. В терминале прописываем:
pip install -r requirements.txt
5. После установки недостающих пакетов прописываем команду запуска:
flask run
6. Открываем в браузере адрес 127.0.0.1:5000
