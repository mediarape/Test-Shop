Приветствую!

Для запуска тестового проекта из среды разработки необходимо прежде всего
инициализировать базу данных MySQL и объявить необходимые переменные среды 
для запуска приложения.

Для инициализации БД я использовал сервер Denwer, поддерживающий MySQL "из коробки". 
Инициализация БД происходит при запуске файла init_sql.py, который также обращается к файлу 
sql.txt,чтобы принять из него записи для таблиц БД (товары и категории).
Обратите внимание, что инициализирующий скрипт обращается к БД "по умолчанию" - 
к "localhost" с пользователем "root" и без пароля. Для инициализации на сервере с паролем необходимо 
изменить соответствующие значения в переменной dbCfg в файле init_sql.py и отредактировать 
переменную app.config['SQLALCHEMY_DATABASE_URI'] в файле __init__.py.

Для установки необходимых плагинов сверяемся с файлом requirements.txt:
>>> pip install -r requirements.txt

Для запуска приложения используем команду
>>> flask run.

Сайт будет запущен на локальном сервере 127.0.0.1:5000
Для поверхностного тестирования и проверки задач реализован минимальный фронтенд.
