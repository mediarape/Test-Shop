Инструкция по запуску приложения Test Shop:

1. Установите Denwer: http://www.denwer.ru
2. Запустите Denwer с помощью ярлыка на рабочем столе
3. Откройте командную строку Windows и перейдите в директорию проекта
4. Введите следующие команды:
python init_sql.py
pip install -r requirements.txt
flask run
5. В браузере откройте 127.0.0.1:5000
